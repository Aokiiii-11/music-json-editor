# 用户操作文档（初版）

## 基本流程
- 粘贴原文 JSON 至顶部“RAW JSON INPUT”。如旧格式包含“英文 | 中文”，系统自动分离并保留原文。
- 在“中文译文 JSON（参考）”文本框粘贴完整中文译文 JSON（结构需与原文一致）。
- 查看匹配诊断：缺失/多余/类型不一致将显示统计与详情。
- 在底部可视化编辑器编辑“原文”内容；右侧显示译文参考，不会写回原文。

## 自动翻译（可选）
- 点击“Auto Translate”将生成参考译文（不修改原文），并填充译文输入区域与诊断。

## 导出
- Export Original：导出当前编辑后的纯净原文 JSON。
- Export Translation：导出当前加载的中文译文 JSON（参考用途）。
- 导出前如存在缺失路径将提示确认，不影响原文导出。

## 注意事项
- 所有符号（包括 `|`、`丨`、`｜`、标点与格式标记）在原文中均保持原样。
- 译文仅作为参考显示；要修改译文请在译文 JSON 输入框中更新并校验结构一致性。
- 若结构不一致（类型不一致/路径缺失），建议修正译文后再导出译文。

## 兼容与迁移
- 旧格式“英文 | 中文”数据会自动拆分：原文保留管道左侧，译文映射由右侧生成。
- 现有编辑操作与布局保持一致，无需额外学习成本。
